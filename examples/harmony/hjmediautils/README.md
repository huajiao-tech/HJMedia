# hjmediautils

hjmediautils属于HJMedia媒体开源的一部分， HJMedia SourceCode: [huajiao-tech/HJMedia · GitHub](https://github.com/huajiao-tech/HJMedia)

## 简介

hjmediautils为hjpusher、hjplayer提供共有的依赖

## 下载安装

`ohpm install @hj-live/hjmediautils`

## 主要功能

* 鸿蒙人脸关键点检测功能：
  - 提供推流器前处理、播放器后处理需要的人脸关键点检测模块封装
